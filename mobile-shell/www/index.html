Esta compilación no apunta a ningún servidor

No estás viendo tus datos. La app no logró cargar el servidor de CARTERA+, así que está mostrando el contenido de emergencia que lleva dentro. Nada de lo que veas aquí son tus finanzas.

Suele pasar por una de dos razones: se compiló en modo empaquetado a propósito (CAP_BUNDLED=1), o el dispositivo no tiene conexión con el servidor.

Cómo arreglarlo

Vuelve a compilar apuntando al servidor y reinstala la app:

cd mobile-shell
npm run sync
npx cap run ios      # o: npx cap run android

El servidor de producción ya es el valor por defecto, así que no necesitas exportar nada. Para probar contra tu máquina en la red local, indica la dirección: 

CAP_SERVER_URL="http://10.0.2.2:3000/m" npm run sync

Si ves esta pantalla en un dispositivo de alguien más, avísale: su app está mal configurada y no está mostrando información real.
